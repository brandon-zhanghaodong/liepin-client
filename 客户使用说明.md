# 招聘工厂 · 猎聘搜人客户端 v1.2

## 安装（只需两步）

### ① 下载对应版本

| 你的 Mac | 下载文件 |
|----------|----------|
| Apple Silicon（M1/M2/M3/M4，2020年及以后） | `招聘工厂-1.0.0-arm64.dmg` |
| Intel 芯片（2020年以前） | `招聘工厂-1.0.0.dmg` |

### ② 安装

1. 双击下载的 .dmg 文件
2. 把 `招聘工厂.app` 拖到 Applications 文件夹
3. 首次打开: 右键 → 打开（或 系统设置 → 隐私与安全性 → 仍要打开）

## 使用

### 第一次运行

1. 打开 `招聘工厂.app`
2. 程序会自动检测环境 → **自动下载 Chromium 浏览器**（约 336MB，仅首次需要）
3. 主窗口是猎聘首页，**扫码登录你的猎聘账号**
4. 登录后，程序打开搜索窗口，输入关键词即可搜索

### 搜索

在弹出的搜索窗口中输入公司+职位，例如：
- `字节跳动 HRD`
- `人工智能 CTO`
- `华为 产品经理`

### 搜索结果

搜索完成后，结果会自动：
- ✅ 推送到 **飞书群**
- ✅ 写入 **飞书 Bitable** 候选人库
- ✅ 保存到 **本地 CSV**（`~/.liepin_client/candidates/`）

## 优势（对比旧版）

| 维度 | 旧版（服务器 API） | 🆕 新版（本地 Playwright） |
|------|-------------------|--------------------------|
| 数据流 | 客户端 → 服务器 → 猎聘 | 客户端 → 猎聘（直连） |
| IP 风险 | 所有客户共用服务器 IP，易被封 | 每个客户用自己的 IP |
| 安装 | 依赖系统 Python，版本不兼容 | 独立 Electron，自带 Node.js |
| 稳定性 | 服务器故障则全部不可用 | 完全本地运行，无单点故障 |
| 包大小 | 依赖 Python 环境 | 112MB（含 Electron） |
