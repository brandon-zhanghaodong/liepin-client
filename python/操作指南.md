# 招聘工厂 · 猎聘搜人客户端 v1.2 — 操作指南

## 安装（1分钟）

### Mac 用户

打开终端（Terminal），复制粘贴下面一整行后回车：

```bash
curl -sSL https://raw.githubusercontent.com/brandon-zhanghaodong/recruitment-factory-client/main/python/install.sh | bash
```

**或者手动安装：**

```bash
# 三步
pip3 install playwright requests
playwright install chromium
```

### Windows 用户

```bash
pip install playwright requests
playwright install chromium
```

---

## 使用

### 第一步：运行

```bash
cd ~/.liepin_client
python3 liepin_cli.py
```

如果觉得每次输路径麻烦，可以设别名（只需要做一次）：

```bash
echo 'alias liepin="cd ~/.liepin_client && python3 liepin_cli.py"' >> ~/.zshrc
source ~/.zshrc
liepin
```

### 第二步：首次登录

程序会打开一个独立浏览器窗口，进入猎聘找简历页面。

- 如果页面显示「请登录」→ **扫码登录你的猎聘账号**（只用第一次）
- 如果已经登录 → 在终端按回车继续

### 第三步：搜索

在终端输入关键词，例如：

```
🔍 关键词 > 字节跳动 HRD
```

程序会自动搜索并翻页提取候选人。

### 第四步：查看结果

搜索结果会：

1. ✅ 在终端显示预览
2. ✅ 推送到飞书群
3. ✅ 写入飞书 Bitable 候选人库
4. ✅ 保存到本地 CSV（`~/.liepin_client/candidates/`）

| 数据 | 去向 |

## 常见问题

**Q: 提示 `command not found: pip3`？**
A: 请先安装 Python 3：https://www.python.org/downloads/

**Q: 下载 Chromium 太慢？**
A: 约 336MB，取决于你的网速，一般 1-3 分钟。第一次运行后才需要下载。

**Q: 搜索时弹出的浏览器影响我工作吗？**
A: 不影响。这是独立的 Chromium 窗口，跟你正在用的 Chrome 完全无关。可以把它缩小到后台。

**Q: 每天都要重新登录吗？**
A: 不需要。登录态会持久化保存在本地，猎聘的 cookie 有效期通常是 30-90 天。

**Q: 搜索到一半报错了？**
A: 可能是猎聘页面改版或者网络波动。重新运行一次即可。
