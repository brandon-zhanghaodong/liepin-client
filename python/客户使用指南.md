# 招聘工厂 · 猎聘搜人客户端 v1.2 — 使用指南

## 一、安装

下载 `招聘工厂_v1.2.dmg`，双击打开，把 `招聘工厂.app` 拖到 Applications 文件夹。

## 二、首次使用

### 第一步：打开应用

双击 `招聘工厂.app` 打开（如果 macOS 提示"无法验证开发者"，请到 **系统设置 → 隐私与安全性** 中点击"仍要打开"）。

### 第二步：自动安装浏览器

第一次打开时，程序会自动下载 Chromium 浏览器（约 336MB），**请保持网络畅通**，下载过程约 1-3 分钟。

> ⚠️ 这个过程只在第一次运行时有，后续启动秒开。

### 第三步：扫码登录猎聘

程序会打开一个独立的浏览器窗口（**不影响你正在用的 Chrome**），自动进入猎聘找简历页面。

- 如果页面显示需要登录 → **扫码登录你的猎聘账号**
- 如果页面已经登录 → 自动继续

登录后，在 GUI 界面输入搜索关键词即可。

### 第四步：开始搜索

在 GUI 窗口输入关键词，例如：
- `字节跳动 HRD`
- `人工智能 CTO`
- `华为 产品经理`

点击「开始搜索」，耐心等待搜索结果。

## 三、后续使用

**以后每次使用：**
1. 打开 `招聘工厂.app`
2. 输入关键词搜索
3. 结果会自动推送到：
   - ✅ 飞书群（通知）
   - ✅ 飞书 Bitable（候选人库）
   - ✅ 本地 CSV 文件

搜索完成后，**不需要手动关闭浏览器**，程序会自己关。

## 四、常见问题

**Q: 第一次打开提示"无法打开"？**
A: 到系统设置 → 隐私与安全性 → 安全性，选择"仍要打开"

**Q: 搜索时说 Chromium 未安装？**
A: 关闭程序重新打开，程序会自动检测并安装。如果卡住，手动在终端运行：
```bash
pip3 install playwright && playwright install chromium
```

**Q: 搜索太慢？**
A: 这是正常的，猎聘搜索结果需要逐页翻取，一次搜索约 30-60 秒。

**Q: 浏览器窗口出现在后台？**
A: 这是独立的 Chromium 浏览器，不会影响你的 Chrome。可以手动把它拉到前台查看进度。

## 五、数据去向

| 数据 | 去向 |
|------|------|
| 候选人信息 | 飞书 Bitable「HR Agent工厂-招聘」表 |
| 搜索通知 | 飞书招聘群 |
| 原始数据 | `~/.liepin_client/candidates/` 目录下的 CSV |
| 登录态 | `~/.liepin_client/liepin_storage.json`（自动保存） |

## 六、服务器端搜索（备用）

如果本机搜索遇到问题，可以联系 Brandon 使用服务器端搜索。服务器端搜索依赖统一 IP，尽量优先使用本地客户端。
